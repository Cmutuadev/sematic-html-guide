Before vs After: Semantic HTML

Compare old HTML with new semantic HTML.

Example 1: Website Structure

❌ Before (Old Way)

<div>
  <div>My Website</div>
</div>

<div>
  <div>My Article</div>
  <div>This is my content...</div>
</div>

<div>Copyright 2024</div>

✅ After (New Way)

<header>
  <h1>My Website</h1>
</header>

<main>
  <article>
    <h2>My Article</h2>
    <p>This is my content...</p>
  </article>
</main>

<footer>
  <p>Copyright 2024</p>
</footer>

Example 2: Navigation Menu

❌ Before (Old Way)

<div>
  <a href="#">Home</a>
  <a href="#">About</a>
</div>

✅ After (New Way)

<nav>
  <a href="#">Home</a>
  <a href="#">About</a>
</nav>

Why Use Semantic HTML?

  • Better for search engines
  • Helps people with disabilities
  • Cleaner and easier to read
